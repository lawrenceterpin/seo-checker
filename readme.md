## Présentation

#### Langages: HTML, CSS, Javascript
Site de démo du plugin Seo Checker avec Node.js

## Prérequis

[GIT](https://git-scm.com/downloads)\

## Installation

Installtion des package Node:
```
npm i
```
Compilation du .scss en .css:
```
npm run sass
```
